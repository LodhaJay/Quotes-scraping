# Quotes-scraping
## Overview
this project scraps quotes authors and tags from quotes.toscrape.com 

## Features 
fetches data from a webpage,  
Extracts quotes authors and tags,  
save the data into CSV amd JSON file.  

## Libraries 
requests, beautifulsoup4, os, pandas

## How to Run
1. Open work.ipynb in VS Code or Jupyter Notebook.
2. Run cells step by step.
3. Final dataset will be saved at:
   CSV/quotes_dataset.csv, 
   CSV/quotes_dataset.json
